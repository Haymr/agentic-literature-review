# Getting Started: Agentic AI Literature Review

Welcome! This beginner-friendly tutorial will show you how to set up and run the multi-agent AI literature review pipeline on your own machine. **You do not need any coding experience to use this.**

## 1. Prerequisites

Before starting, ensure you have **n8n** running on your computer. If you haven't installed it yet, the easiest way is via Docker or the Desktop App.
Make sure you know where your n8n local files directory is. For Docker installations, this is commonly mapped to `~/.n8n/local-files/` (or inside the container at `/home/node/.n8n-files/`).

## 2. Importing the Workflows

The system is composed of 5 different workflow files located in the `workflows/` folder. You must import them into your n8n workspace:
1. Open your n8n workspace in your web browser.
2. Create a new workflow, click the **Options (three dots)** menu in the top right, and select **Import from File**.
3. Import the workflows one by one in the correct order:
   - `02-paper-extractor.json`
   - `03-thematic-analyzer.json`
   - `04-review-writer.json`
   - `05-fact-checker.json`
   - `01-main-supervisor.json`
4. **Important:** After importing, make sure all 5 workflows are saved and set to **Active**.

## 3. Preparing Your Documents

The agents need academic papers to read.
1. Gather the PDF files you want to include in your literature review.
2. Place all the PDF files into the local n8n files folder. For most typical setups, this is the root directory mapped to `/home/node/.n8n-files/` in docker. (Ensure there are no other random PDFs in the folder to avoid confusing the agents).

## 4. Running the Pipeline

1. In n8n, open the `01-main-supervisor` workflow.
2. Ensure the workflow is in test mode by clicking the **Test Workflow** button at the bottom.
3. Open the **Chat** interface within the n8n window.
4. Type `Start` (or `Başla`) and hit enter.
5. Sit back and watch! You will see the Supervisor Agent automatically call the other tools one by one.

## 5. Getting Your Results

Because the pipeline writes files directly to your hard drive, you don't need to copy-paste messy text from the chat window.
1. Once the Supervisor says the pipeline is complete, navigate back to your local n8n files directory.
2. You will find several new files generated by the AI:
   - `temp_extracted.json`: The raw text data pulled from the PDFs.
   - `temp_analysis.json`: The thematic analysis.
   - `literature_review_output.txt`: The JSON Fact Check report proving the text is accurate.
   - **`literature_review.html`**: The final beautifully-styled Literature Review document.

**How to get a PDF:** Simply double-click `literature_review.html` to open it in Chrome or Safari, press `Ctrl+P` (or `Cmd+P`), and select "Save as PDF". You now have a complete, offline, formatted IEEE literature review.
